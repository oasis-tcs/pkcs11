## FASTHASH

### Definitions

Mechanisms:

- CKM_FASTHASH

### FASTHASH digest

The FASTHASH mechanism, denoted **CKM_FASTHASH**, is a mechanism for message
digesting, following the U.S. government’s algorithm.

It does not have a parameter.

Constraints on the length of input and output data are summarized in the
following table:

| Function | Input length | Digest length |
|----------|--------------|---------------|
| C_Digest | Any          | 40            |
table: FASTHASH: Data Length
